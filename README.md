# Garden Horizons API

Free to use API for getting predictions, stock data, weather data, and more.

---

## Main Page
[https://grhz.onrender.com/](https://grhz.onrender.com/)

---

## API Endpoints

| Endpoint       | URL                                           | Description                        |
|----------------|-----------------------------------------------|------------------------------------|
| Stocks         | [api/stocks](https://grhz.onrender.com/api/stocks)         | Get current stock data             |
| All Items      | [api/all](https://grhz.onrender.com/api/all)               | Get all data                      |
| Predictions    | [api/predictions](https://grhz.onrender.com/api/predictions) | Get the predictions for stock              |
| Weather        | [api/weather](https://grhz.onrender.com/api/weather)       | Get the current weather data           |
